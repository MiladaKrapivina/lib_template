# Library Management

Учебная информационная система для управления библиотечным фондом на Python.

## Возможности

- CRUD-операции для книг.
- Регистрация и редактирование читателей.
- Выдача, возврат, расчёт штрафа за просрочку.
- Просмотр истории выдач по книге и читателю.
- CLI с ролями библиотекаря и читателя.
- Персистентность в JSON-файлах каталога `data/`.

## Запуск

```bash
python -m src.main
```

## Тесты

```bash
python -m unittest discover
```
